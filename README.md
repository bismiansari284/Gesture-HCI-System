\# Gesture Controlled Human Computer Interaction System



\## Project Description



This project is a Gesture Controlled Human Computer Interaction System that allows users to control a computer using hand gestures.



The system uses Computer Vision and Machine Learning to detect hand gestures in real-time and perform actions like mouse movement and system control.





----------------------------------



\## Features



\- Real-time hand tracking

\- Gesture recognition

\- Mouse cursor control

\- Click operations

\- Volume control

\- Brightness control





----------------------------------



\## Technologies Used



\- Python

\- OpenCV

\- MediaPipe

\- NumPy

\- Machine Learning
